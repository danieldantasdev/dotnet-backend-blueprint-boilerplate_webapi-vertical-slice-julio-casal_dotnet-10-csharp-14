# dotnet-backend-blueprint-boilerplate_webapi-vertical-slice_dotnet-10-csharp-14
Blueprint de backend em .NET 10 para APIs REST production-ready, baseado em Vertical Slice Architecture, com PostgreSQL e EF Core, autenticação via Keycloak, Swagger autenticado, tratamento global de erros, CI/CD com GitHub Actions, Aspire para ambiente local e suporte a deploy em Azure.
